# 📚 Fanfics Archive Bot

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)](https://python.org)
[![aiogram](https://img.shields.io/badge/aiogram-3.x-green)](https://github.com/aiogram/aiogram)
[![SQLite](https://img.shields.io/badge/SQLite-3-lightgrey)](https://sqlite.org)

Телеграм-бот для поиска и архивирования фанфиков.
Позволяет находить файлы по названию, автору, рейтингу и меткам, загружать новые произведения и просматривать статистику.

**Проект создан в рамках обучения Python-разработке.**

## ✨ Функциональность

- 🔎 **Поиск** по названию, автору и тексту (FTS)
- ⭐ **Расширенный поиск** с фильтрами по рейтингу, размеру и меткам
- 📤 **Загрузка файлов** в бота для пополнения архива
- 📊 **Статистика** архива: количество фиков, рейтинги, популярные метки
- 🧠 **Парсинг метаданных** из шапок файлов (рейтинг, размер, метки)
- 🗂️ **Пагинация** результатов с инлайн-кнопками

## 🛠 Стек технологий

| Технология       | Описание                              |
|------------------|---------------------------------------|
| Python 3.10+     | Основной язык разработки              |
| aiogram 3.x      | Фреймворк для создания Telegram ботов |
| SQLite + FTS5    | База данных с полнотекстовым поиском  |
| python-dotenv    | Загрузка переменных из `.env`         |
| python-docx      | Чтение метаданных из docx файлов      |
| ebooklib + bs4   | Парсинг epub и fb2                    |
